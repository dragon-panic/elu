## Pattern to follow

`crates/elu-cli/src/cmd/search.rs` — the only CLI command that currently talks to a registry. Shape:

1. Resolve registry URL from `ctx.registry` with `DEFAULT_REGISTRY` fallback (reject if `ctx.offline`).
2. Construct `RegistryClient::from_env_str(...)`.
3. Open the store from `ctx.open_store()?`.
4. Parse `args.reference` into a `PackageRef` (use `crate::refs_parse::parse_ref`).
5. Build a tokio current-thread runtime; `rt.block_on(publish_package(...))`.
6. Emit result: JSON object on `--json`, human line otherwise.
